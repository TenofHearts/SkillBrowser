# Academic Abstract Writer

## Overview

Use this pattern to draft or revise an academic abstract that is concise, specific, and easy to scan. The abstract should state the research gap, method, result, and implication.

## Procedure

1. Identify the field context and the unresolved gap.
2. State the paper's aim or central claim.
3. Name the method or evidence source.
4. Summarize the strongest result with concrete detail.
5. End with the implication, contribution, or use case.
6. Remove unsupported novelty claims, vague adjectives, and citations unless requested.

## Output Format

Provide one polished abstract. If the user requested revision, optionally include a short note listing the main edits.

## Examples

For a 150-word limit, prioritize gap, method, result, and implication over background detail.

## Failure Modes

Avoid writing a teaser without results. Avoid overclaiming beyond the supplied evidence.
